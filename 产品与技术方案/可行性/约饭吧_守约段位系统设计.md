# 约饭吧：守约段位系统设计文档

> 创意来源：用户灵感
> 核心理念：把守约行为游戏化，像打排位赛一样经营自己在朋友圈里的"信用战绩"
> 设计原则：轻量、有趣、不强迫、天然可传播

---

## 一、为什么要做段位系统

普通守约率是一个冷冰冰的数字（92%）。  
段位是一个**有情感重量的身份标签**（黄金守约者）。

区别在于：
- 数字让人理性评估
- 段位让人产生**荣誉感和危机感**

荣誉感让用户主动分享，危机感让用户主动维护。这两个情绪是产品传播和留存的核心燃料。

---

## 二、段位体系设计

### 2.1 段位等级（从低到高，共7级）

| 段位 | 名称 | 守约率门槛 | 图标 | 定义 |
|------|------|-----------|------|------|
| Ⅰ | 新人 | 不足3次记录 | 🌱 | 还在建立记录中 |
| Ⅱ | 青铜 | < 60% | 🥉 | 经常放鸽子，圈子里有名的不靠谱 |
| Ⅲ | 白银 | 60–74% | 🪙 | 时好时坏，朋友约你要做好备份计划 |
| Ⅳ | 黄金 | 75–84% | 🥇 | 基本靠谱，偶尔会消失 |
| Ⅴ | 铂金 | 85–92% | 💠 | 高可靠，朋友约饭优先考虑的人 |
| Ⅵ | 钻石 | 93–98% | 💎 | 说到做到，圈子里的定海神针 |
| Ⅶ | 传奇 | 99–100% + 最少20次 | 👑 | 从未爽约，活着的都市传说 |

### 2.2 段位计算逻辑

```
基础守约率 = 实际到场次数 / 报名总次数

段位分（Score）= 基础守约率 × 权重系数
```

**权重系数说明：**
- 近6次聚餐权重 × 1.5（近期表现更重要）
- 6次以前的记录权重 × 1.0
- 目的：允许人改变，不让早期爽约永久拖累

**降段机制：**
- 连续2次爽约，段位直接降1级
- 给用户强烈的"我要挽回"冲动

**升段机制：**
- 连续5次守约，段位升1级
- 给用户明确的努力方向

## 三、战绩卡片系统（核心传播机制）

### 3.1 个人战绩卡片

类似游戏里的战绩截图，可一键生成，分享到朋友圈/微信群。

```

│                                      │
│  张三的约饭战绩                        │
│  ────────────────                    │
│  守约率      96%                      │
│  总参与      24 局                    │
│  连续守约    🔥 7 次                  │
│  最长连续    🏆 12 次                 │
│                                      │
│  朋友评价                             │
│  "约他从来不会被放鸽子" — 李四         │
│                                      │
│  [用约饭吧记录你的守约战绩]             │
└─────────────────────────────────────┘
```

**分享触发时机（自动提示）：**
- 段位升级时："恭喜升入铂金，分享给朋友？"
- 达成新连续守约记录时
- 年度总结时（12月31日）
- 守约率达到整数关口（90%、95%、100%）

## 四、朋友圈排行榜

### 4.1 小圈子排行（最核心）

不做全平台排行（没意义），只做**你常约饭的小圈子**内部排行。

在"常聚小组"里，显示：

```
你们这个圈子的守约榜
────────────────────
🥇 No.1  王芳   👑 传奇   100%
🥈 No.2  张三   💎 钻石    96%
🥉 No.3  李四   💠 铂金    88%
    No.4  小明   🥇 黄金    76%
    No.5  老王   🥉 青铜    55%  ← 你
```

**你在自己最在乎的人面前排最后，这个压力是真实的。**

### 4.2 好友间的"挑战"机制

- 钻石用户可以向铂金用户发出"守约挑战"
- 接受挑战后，双方下一次约饭的守约情况会被特别标记
- 纯虚拟、无奖惩，但有仪式感和竞争欲

## 五、段位系统与内购的结合

### 免费版
- 显示自己的段位和守约率
- 看到圈子排行（排名模糊处理，只显示自己的位置）
- 生成基础版战绩卡片（无自定义样式）

### 付费版（解锁）
- 完整圈子排行，所有人排名可见
- 高级战绩卡片样式（3套主题：极简/炫酷/温暖）
- 年度守约报告（像Spotify年度总结一样）
- 段位徽章展示在约饭局详情页

**付费逻辑：** 免费版让用户感受到排行的存在，但看不到完整榜单——这个"残缺感"是转化的最强驱动力。

## 六、年度守约报告（年终传播核弹）

每年12月底，生成个人年度报告，类似Spotify Wrapped：

```
2026 你的约饭年度报告

参与约饭    18 局
守约战绩    17/18  守约率 94%
最常约饭    李四（6次在一起）
最远聚餐    上海·外滩餐厅
段位变化    白银 → 铂金  📈
年度评价    "你是圈子里最靠谱的人之一"

你的守约比全国 87% 的用户更高
```

这个报告**必然会大规模传播**，是约饭吧全年最大的自然营销事件。

---

## 七、防滥用设计

段位系统存在的潜在问题：发起人恶意标记他人爽约。

**应对机制：**
- 爽约标记需要**超过50%参与人确认**才生效（不是发起人单方面决定）
- 被标记者可以在24小时内提交"申诉"（填写原因，其他人投票）
- 连续被申诉成功3次的发起人，其标记权限降低

---

## 八、总结：段位系统对产品的价值

| 维度 | 原产品 | 加入段位系统后 |
|------|--------|----------------|
| 用户留存理由 | 约饭工具，用时打开 | 维护段位，主动回访 |
| 传播机制 | 分享链接邀朋友 | 晒战绩卡片，被动带新用户 |
| 付费转化点 | 功能解锁 | 功能+荣誉双驱动 |
| 社交货币 | 无 | 高段位=熟人圈子里的身份标签 |
| 差异化壁垒 | 中 | 高（数据积累形成迁移成本） |

**段位系统把约饭吧从一个"有用的工具"变成了一个"有面子的游戏"。**  
这两者的用户粘性，不在同一个量级。
