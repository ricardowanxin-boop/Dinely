# DineRank / 食否 隐私政策（网页版文案）

最近更新：2026-04-03

## 1. 我们是谁

`DineRank / 食否 / Dinely` 是一款帮助熟人组织约饭、投票选择时间与餐厅、查看当天地图、签到、AA 分摊和生成守约战报的 iPhone 应用。

## 2. 我们处理哪些数据

应用会在当前设备上保存以下必要数据：

- 草稿与已创建的约饭局
- 参与人昵称与头像 Emoji
- 时间投票与餐厅投票
- 签到状态、AA 结算结果、守约战报
- 主题与偏好设置

我们不要求你注册开发者自有账号，也不做广告追踪。

## 3. 云端协作

当你创建可分享的约饭局时，为了让受邀参与者加入同一场饭局，相关协作数据会通过 Apple CloudKit 同步，包括：

- 饭局标题
- 参与人昵称
- 头像 Emoji
- 时间投票
- 餐厅投票
- 签到状态
- 位置共享开关

这些数据用于多人协作，不会出售给广告平台，也不会用于广告画像。

## 4. 定位与地图

应用仅在约饭当天地图与自动签到场景中使用定位权限，用于：

- 显示你距离餐厅还有多远
- 支持自动签到

地图展示与餐厅搜索使用 Apple MapKit；在中国大陆，地图底图可能显示高德地图署名。

## 5. 购买与支付

当前版本仅提供一次性买断 Pro，由 Apple App Store 负责支付、恢复购买和账单处理。我们不会接触你的银行卡信息、Apple ID 密码或支付凭证原文。

## 6. 你可以如何管理数据

你可以通过以下方式管理或删除数据：

- 删除 App 以清除本地缓存数据
- 在系统设置中撤回定位或通知权限
- 在 Apple ID 中管理或恢复购买
